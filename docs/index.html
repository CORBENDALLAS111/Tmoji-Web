TMoji-Web

Production-ready library for rendering Telegram Premium Custom Emojis on any website. Supports static PNG, animated TGS (Lottie), WEBM, and GIF formats with lazy loading and intelligent caching.

What is this?

TMoji-Web bridges the gap between Telegram's premium custom emoji ecosystem and the open web. It provides a complete frontend library, backend API service, and documentation to seamlessly integrate Telegram's animated emoji packs into your web applications.

Key Features

  • Multiple Formats — PNG, TGS (Lottie), WEBM, GIF support
  • Lazy Loading — Emojis load only when visible
  • Smart Caching — Memory + localStorage LRU cache
  • Tree Shakable — Import only what you need
  • TypeScript — Full type definitions included
  • CDN Ready — Works with any static host

Quick Start

1. Initialize the Library

import { TMoji } from 'tmoji-web';

const tg = new TMoji({
  apiBaseUrl: 'https://api.yourdomain.com',
  defaultSize: '1.2em',
  animated: true
});

2. Load an Emoji Pack

const pack = await tg.loadPack('https://t.me/addemoji/vector_icons_by_fStikBot');
console.log(`Loaded ${pack.stickers.length} emojis`);

3. Render a Single Emoji

// By ID
await tg.renderTo('#emoji-container', '123456789');

// With options
await tg.renderTo('#emoji-container', '123456789', {
  size: '2em',
  animated: true
});

4. Render Text with Emojis

await tg.renderText('#message', 'Hello {emoji:123456789} World!');

5. Auto-parse HTML Tags

<!-- In your HTML -->
<p>Check this out: <tg-emoji id="123456789" size="1.5em"></tg-emoji></p>

<!-- In your JS -->
await tg.parseAll();

TMojiClass

Constructor

new TMoji(config?: Partial<TMojiConfig>)

Configuration Options

Property Type Default Description
apiBaseUrl string http://localhost:8000 Backend API base URL
defaultSize string 1.2em Default emoji size
animated boolean true Enable animations by default
cacheDuration number 86400000 Cache duration in ms (24h)
lazyThreshold string 50px Lazy loading margin

Methods

loadPack(urlOrName: string)

Load an emoji pack from Telegram link or pack name.

const pack = await tg.loadPack('adaptive1pack_by_TMojiBot');
// or
const pack = await tg.loadPack('https://t.me/addemoji/vector_icons_by_fStikBot');

getEmoji(id: string)

Fetch a single emoji by its custom emoji ID.

const emoji = await tg.getEmoji('123456789');
console.log(emoji.file_type); // 'tgs', 'png', 'webm', etc.

renderTo(container, emojiId, options?)

Render an emoji to a DOM element.

await tg.renderTo('#container', '123456789', {
  size: '2em',
  animated: true,
  onLoad: () => console.log('Loaded!')
});

lazyRender(container, emojiId, options?)

Render with lazy loading (waits until visible).

await tg.lazyRender('#container', '123456789');

renderText(container, text, options?)

Parse and render text with {emoji:ID} syntax.

await tg.renderText('#message', 'Hello {emoji:123} World {emoji:456}!');

parseAll(options?)

Auto-parse all <tg-emoji> tags in the document.

await tg.parseAll();

Backend Setup

Prerequisites

  • Python 3.9+
  • Telegram Bot Token (get from @BotFather)

Installation

cd backend
pip install -r requirements.txt

Configuration

Set environment variable:

export BOT_TOKEN="your_bot_token_here"

Running

python -m app.main

Or with uvicorn directly:

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
Important

The backend requires a valid Telegram Bot Token. Without it, the API will return 503 errors. The bot does not need any special permissions, but it must be active.

Animated Emojis (TGS/Lottie)

What is TGS?

TGS is Telegram's sticker format — a gzip-compressed Lottie JSON animation. The library automatically decompresses TGS files and renders them using lottie-web.

Rendering Modes

Format Renderer Notes
TGS lottie-web (SVG) Decompressed via CompressionStream API
WEBM <video> element Hardware accelerated
GIF <img> element Universal support
PNG <img> element Static fallback

Performance Tips

  • Use lazyRender() for emojis below the fold
  • Set animated: false for low-power devices
  • Limit concurrent Lottie animations (max 10-20)
  • Use PNG thumbnails as placeholders

Performance

Lazy Loading

Emojis use Intersection Observer to load only when visible. Configure the threshold in constructor options.

Caching Strategy

Two-tier caching system:

  • Memory Cache: LRU cache (max 100 items)
  • localStorage: Persistent across sessions
